# MossGPT

基于 gpt-3.5-turbo 的 utools 插件

欢迎 start ✨

[![ppmBXPs.png](https://s1.ax1x.com/2023/03/08/ppmBXPs.png)](https://imgse.com/i/ppmBXPs)

## 安装

首先需要下载安装 utools：https://www.u.tools/

然后进入 utools 的 `插件市场` 搜索 `chatgpt` 找到插件进行安装
