# MossGPT

基于 gpt-3.5-turbo 的 utools 插件

欢迎 start ✨

[![pptFBSU.png](https://s1.ax1x.com/2023/03/19/pptFBSU.png)](https://imgse.com/i/pptFBSU)

## 安装

首先需要下载安装 utools：https://www.u.tools/

然后进入 utools 的 `插件市场` 搜索 `mossgpt` 找到插件进行安装
